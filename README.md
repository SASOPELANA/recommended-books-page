# Libros Recomendados

![BOOKS](./public/img/books.webp)

## Descripción

Este es un sitio web de recomendación de libros construido con [Astro](https://astro.build/). El sitio muestra una lista de libros recomendados, divididos en categorías. Cada libro tiene su propia página de detalles con información adicional.

## Características

-   **Sitio Estático:** Generado con Astro para un rendimiento óptimo y SEO.
-   **Dos Categorías de Libros:** Libros generales y libros para desarrolladores.
-   **Páginas de Detalle:** Cada libro tiene una página dedicada con más información.
-   **Estilos con Tailwind CSS:** Interfaz de usuario moderna y personalizable.
-   **Contenido con Markdown/MDX:** Fácil de gestionar y añadir nuevos libros.
-   **Componentes Interactivos:** Uso de Astro Islands para componentes dinámicos en el cliente.

## Tecnologías

-   [Astro](https://astro.build/)
-   [Tailwind CSS](https://tailwindcss.com/)
-   [MDX](https://mdxjs.com/)
-   [Vercel](https://vercel.com/) (para despliegue)
