## Descripción

Este superventas mundial es una guía completa al lenguaje de programación **Python**. Gracias a esta introducción clara y práctica, el lector podrá comenzar rápidamente a escribir programas, resolver problemas y desarrollar aplicaciones funcionales.

Esta **tercera edición actualizada** ha sido revisada en profundidad para reflejar las últimas novedades del lenguaje. Incluye la incorporación de **VS Code** como editor de texto, el uso del módulo **pathlib** para la gestión de archivos y **pytest** para la realización de pruebas, así como las actualizaciones más recientes en **Matplotlib**, **Plotly** y **Django**.

El aprendizaje comienza con los conceptos fundamentales de la programación, como _variables_, _listas_, _clases_ y _bucles_, acompañados de ejercicios prácticos diseñados para fomentar la escritura de código limpio y comprensible. El lector también aprenderá a crear programas interactivos y a probar su código de forma segura antes de integrarlo en proyectos reales.

En la segunda parte del libro, los conocimientos adquiridos se aplican en tres proyectos prácticos:

- Un juego arcade inspirado en **Space Invaders**
- Un conjunto de **visualizaciones de datos** utilizando bibliotecas populares de Python
- Una **aplicación web sencilla** que puede desplegarse en línea

Si desea profundizar en la programación, este libro le ayudará a escribir programas reales de forma rápida y eficaz.  
¿Por qué esperar más? **Empiece ya a programar.**
